# SortBy

Which response parameter to sort by. **Possible values**: `api_key`, `created_at`, `name`, `enabled`.


## Example Usage

```typescript
import { SortBy } from "@cloudinary/account-provisioning";

let value: SortBy = "created_at";
```

## Values

```typescript
"api_key" | "created_at" | "name" | "enabled"
```
